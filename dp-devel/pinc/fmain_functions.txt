Shorthand functions to replace the current mysql_xyz style functions.

ptog($varNM)
     Send variable name ($varNM) to ptog() to retrieve (return) the POST value from form.

function dquery($qstring)
     Replaces mysql_query(query string).  Returns the result resource.

function nrows($res)
     Replaces mysql_num_rows(result resource).  Send result from database query as $res.
     Returns the number of rows returned by the query.

function nfields($res)
     Replaces mysql_num_fields(result resource).  Send result from database query as $res.
     Returns the number of table fields returned by the query.

function arows()
     Replaces mysql_affected_rows().  Sends result from last query.
     Returns the number of rows affected by the last query if it is an INSERT, UPDATE,
     or DELETE query.  Returns -1 if no rows affected.

function derror()
     Replaces mysql_error().  Returns the error text from the last database query or an
     empty string '' if no error exists.

function fresult($res)
     Replaces mysql_free_result(result resource).  Send result from database query as $res.
     Returns true or false according to whether the query result was freed from memory.

function dresult($res,$row,$field)
     Replaces mysql_result(result resource , row index , field name). Send result from 
     database query as $res, row index number as $row, and field name as $field.
     Returns the value of the field specified in $field from the row $row from the query
     result $res.

function drow($res)
     Replaces mysql_fetch_row(result resource).  Send result from database query as $res.
     Returns the result row as an array or false if no more rows exist.

function darray($res)
     Replaces mysql_fetch_array(result resource).  Send result from database query as $res.
     Returns the result row as an array or false if no more rows exist.

function dassoc($res)
     Replaces mysql_fetch_assoc(result resource).  Send result from database query as $res.
     Returns the result row as an associative array or false if no more rows exist.

function rowsquery($qString)
     Send query string as $qString.  Returns the number of rows in the query result.  Use
     when you _only_ need to know the number of rows and will not require a result resource.
